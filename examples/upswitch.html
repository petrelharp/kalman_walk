Brownian motion on an upswitch

Peter Ralph and Josh Shiffman

2017-06-13

Model

Consider the dynamical system given by \[\begin{aligned} \dot{x}(t) &= Ax(t) + Bu(t) \\ y(t) &= Cx(t) \end{aligned}\] where \[\begin{aligned} B^{T} = \begin{bmatrix} 1 & 0 \end{bmatrix}, \qquad C = \begin{bmatrix} 0 & 1 \end{bmatrix} \end{aligned}\] and \[\begin{aligned} A = \begin{bmatrix} -1 & 0 \\ 1 & 0 \end{bmatrix} . \end{aligned}\] This system has impulse response function \[\begin{aligned} h(t) &= C e^{tA} B \\ &= 1 - e^{-t} . \end{aligned}\] The set of equivalent \(A\) are given by \(\{A(\tau): \tau \neq 1 \in \R\}\), where \[\begin{aligned} V(\tau) &= \begin{bmatrix} 1 & \tau \\ 0 & 1 \end{bmatrix} \\ A(\tau) &= V(\tau) A V(\tau)^{-1} \end{aligned}\] These systems are input-output equivalent (although may have different noise properties).

A0 <- matrix(c(-1,1,0,0), nrow=2)
B <- matrix(c(1,0), ncol=1)
C <- matrix(c(0,1), nrow=1)
V <- function (tau) { matrix(c(1,0,tau,1), nrow=2) }
Vinv <- function (tau) { matrix(c(1,0,-tau,1), nrow=2) }
A_fn <- function (tau) { V(tau) %*% A0 %*% Vinv(tau) }
h <- function (t,tau=0,A=A_fn(tau)) { sapply(t, function (tt) C %*% expm(tt*A) %*% B) }
optimal_h <- function (t) { 1 - exp(-t) }

We can define the dynamics in two ways: extrinsically or intrinsically. Extrinsically, we look at the dynamics of a population mean in \(\R^{2\times 2}\), thinking about \(A\) as evolving. Intrinsically, we move about only on the set of equivalent \(A\), letting \(\tau\) evolve. As the population size goes to infinity, the extrinsic dynamics should approach the intrinsic dynamics.

Extrinsic dynamics

Let \(N\) denote the population size, and suppose that at any time the population is spread about from the mean \(A\) as a Gaussian with variance \(\sigma^2\), independently in each entry of \(A\). Then we expect that the population mean takes steps of size \(\sigma/\sqrt{N}\) with a restoring force towards the set of optimal \(A\).

Fitness of hybrids

First we need the fitness function. This will be \(\exp(-D^2/2\sigma^2)\), where \(D\) is the distance to optimal: $$\begin{aligned} D(A) := _0e{-t/4} | h(t,A) - h_0(t) | dt , \end{aligned} where \(h_0(t)\) is the optimal impulse response function.

Df <- function (A,t) {
    exp(-t/(4*pi)) * ( h(t,A=A) - optimal_h(t) )^2
}
D <- function (A, upper=10, ...) {
    f <- function (t) { Df(A,t) }
    integrate(f, lower=0, upper=upper, ...)$value
}

To check this works, the distance of \(A(2)\) to optimal should be zero:

We also need to be able to produce F1s and F2s:

F1 <- function (tau1,tau2) { (A_fn(tau1)+A_fn(tau2))/2 }

gamete <- function (X,Y,rows) { 
    out <- X
    out[rows,] <- Y[rows,]
    return(out)
}

F2 <- function (tau1, tau2, rows1, rows2) {
    (gamete(A_fn(tau1), A_fn(tau2), rows1) + gamete(A_fn(tau1), A_fn(tau2), rows2))/2
}

random_F2 <- function (tau1, tau2) {
    rows1 <- (rbinom(2,size=1,prob=1/2)>0)
    rows2 <- (rbinom(2,size=1,prob=1/2)>0)

    F2(tau1, tau2, rows1, rows2)
}

all_F2s <- function (tau1, tau2) {
    all_rows <- as.matrix(do.call( expand.grid, list(c(TRUE,FALSE))[rep(1,4)] ))
    lapply(1:nrow(all_rows), function (k) {
                x <- all_rows[k,]
                F2(tau1, tau2, x[1:2], x[3:4])
            })
}

dvec <- seq(0, .02, length.out=20)
D_F1 <- sapply(dvec, function (dtau) {
            D(A=F1(tau1=2, tau2=2+dtau))
        } )

D_F2 <- do.call(cbind, mclapply(dvec, function (dtau) {
            F2s <- all_F2s(2, 2+dtau)
            sapply(F2s, D, upper=100)
        }, mc.cores=detectCores() ) )
layout(t(1:2))
plot( dvec, D_F1, type='l', xlab="d tau", ylab="phenotype distance (D)", main="distance of F1s")
